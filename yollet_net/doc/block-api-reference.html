params.order integer 목록 정렬 순서(생성시간) 0:desc, 1:asc

Name Type Constraints Description

result.resultData string 결과 데이터
Errors
Code Message Description
-32000 block api error block api error

{
  "jsonrpc": "2.0",

    "address": "49e7f13db764675275761487a74e3503fafbf58342e5bd74479bd1d0dd8bdf8f",
    "tokenId": "ABC",
    "fromDate": "2021-03-14T13:01:01",
